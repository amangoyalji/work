About the Repository

Monuments image Repository is a collection of images with different categories of India. This repository is useful when any researcher want to do some innovative research on these types of images. it can be useful for the students, educators, and researchers all over the world whose primary research concern is Indian monument image dataset. This dataset contain a11 classes of different nbspcategories of the Indian monuments images. It is a perfect place where each category of images contains different types of orientation of the image. Every category contain at least 100 images of each class. Every user can have chance to select dataset category that will be really interesting for their research. The search is simple here, any user can browse the data sets directly. This repository is only containing monument images data. Some famous images are TajMahal, Qutub-Minar, India-Gate etc.
